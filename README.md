Название проекта: CRUD MVC Spring на Java

Описание

Это веб-приложение реализует функционал CRUD (Create, Read, Update, Delete) для работы с клиентами и заказами. Проект построен с использованием паттерна MVC (Model-View-Controller) на платформе Spring Boot.

Технологии

- Spring Boot: Основной фреймворк для разработки приложения.
- JPA (Java Persistence API): Используется для работы с базой данных.
- Docker: Контейнеризация приложения, работа с PostgreSQL.
- Amplicode: Генерация кода и упрощение разработки.
- Maven: Управление зависимостями и сборка проекта.
- Lombok: Упрощение работы с Java-классами, уменьшение шаблонного кода.
- Spring Test: Тестирование компонентов приложения.
- Spring OpenAPI: Документация API.
- Json-unit: Тестирование JSON-ответов.
- MapStruct: Преобразование объектов между различными типами.

Установка и запуск

1. Клонировать репозиторий:

   git clone <url-репозитория>
   cd <каталог-проекта>


2. Сборка проекта:

   mvn clean install


3. Запуск Docker-контейнера с PostgreSQL:

   docker-compose up


4. Запуск приложения:

   mvn spring-boot:run


Использование

После запуска приложения оно будет доступно по адресу http://localhost:8080.

- Создание клиента: POST запрос на /api/clients
- Получение клиентов: GET запрос на /api/clients
- Обновление клиента: PUT запрос на /api/clients/{id}
- Удаление клиента: DELETE запрос на /api/clients/{id}

Тестирование

Для запуска тестов выполните команду:
mvn test


Документация API

Документация доступна по адресу: http://localhost:8080/v3/api-docs
